# 个人网站 UI 设计需求

## 项目背景

个人主页 + 博客网站，技术栈 Astro，部署 GitHub Pages。目标用户是访问者，不是开发者。

## 目标受众

- 内容读者（公众号读者，来看文章）
- 技术同行（看工程审美和内容质量）
- 潜在合作方（商务信任感）

核心印象：**专业、可信、有积累**

## 风格定义

- 浅色主导 + 深色 Hero 区
- 审美参照：Stripe × Vercel，克制、专业、有秩序感
- 主背景：暖灰浅色
- Hero 区背景：深蓝（`#0f1b2d` 一类）
- 强调色：纯深蓝（`#1d4ed8` 一类）
- 不堆渐变特效，靠排版秩序和细节建立层次感

## 需要设计的页面（共 3 个）

### 1. 主页（Home）

包含：
- 顶部导航：Logo/姓名 + 导航链接（博客、关于、联系）
- Hero 区：名字 + 一句话介绍，深蓝背景
- 最新文章列表：3-5 篇，卡片式
- 页脚：邮件 / GitHub / 公众号链接

### 2. 博客列表页（Blog）

包含：
- 页面标题 + 文章总数
- 文章列表：标题 + 日期 + 摘要，时间倒序
- 分页控件

### 3. 文章详情页（Post）

包含：
- 文章标题 + 发布日期 + 预计阅读时间
- 正文区域（含代码块样式）
- 底部：上一篇 / 下一篇导航

## 交互规范

- 卡片 hover：边框出现深蓝描边
- 导航栏：滚动后背景加毛玻璃模糊效果
- 按钮/链接高亮色：`#1d4ed8`

## 排版规范

- 字体：标题 + 正文用 Inter，代码用 JetBrains Mono
- 正文行高：1.8
- 文章正文最大宽度：720px

## 交付物

- 3 个页面的视觉稿（Desktop 优先，Mobile 次之）
- 标注色值、字号、间距
- 组件状态：默认 + hover（卡片、按钮、导航链接）
